# Python File Organizer GUI

This is a simple Python GUI project that automatically organizes files into folders.

## Features
- Organizes files by type (Images, Videos, Documents, Music)
- Simple GUI interface
- Beginner friendly Python project

## How to Run
1. Install Python 3
2. Download this project
3. Run this command:
python file_organizer_gui.py
